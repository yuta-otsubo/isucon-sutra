# ISURIDE Frontend

## Routes

- **/client**
  - ユーザー側が使用するアプリケーションルート
- **/client/history**
  - 履歴
- **/client/contact**
  - 問い合わせ
- **/driver**
  - ドライバー側(配椅子側)が使用するアプリケーションルート


## Setup

```sh
npx create-remix@latest --template remix-run/remix/templates/spa
```

## Development

```sh
npm run dev
```

## Format

```sh
npm run fmtcheck # format check
npm run fmt # format fix
```

## Production

```sh
npm run build
```

## Codegen

```sh
npm run codegen
```
